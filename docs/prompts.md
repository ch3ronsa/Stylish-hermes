# Master Prompts

## 1. Demo Analysis

```text
Analyze this image briefly: aesthetic, colors, occasion, wearable version.
```

## 2. Demo Transformation

```text
Transform this look into a smart-casual everyday outfit for Istanbul spring. Keep the vibe, but make it practical, wearable, and visually different from the original.
```

## 3. Demo Generation

```text
Generate the transformed version as a realistic fashion image.
```

## 4. Everyday Adaptation

```text
Transform this look into a practical everyday outfit. Keep the same vibe, but make it simpler, more wearable, and suitable for city life.
```

## 5. Office Adaptation

```text
Transform this look into a smart-casual version for a work meeting. Keep the aesthetic direction, but change the pieces so it fits a real office setting.
```

## 6. Budget Adaptation

```text
Transform this look into a budget-friendly version under $150 total. Keep the mood and palette, but replace the pieces with cheaper, realistic alternatives.
```

## 7. Identity Shift

```text
Transform this look into a masculine version. Keep the same aesthetic energy, but redesign the outfit completely for a menswear silhouette.
```

## 8. Initial Profile Setup

```text
Use the ai-personal-stylist skill. Ask me only the minimum questions needed to build a short personal style profile. Then write a short summary suitable for Hermes memory. Do not store my full wardrobe in memory.
```

## 9. Inspiration Analysis

```text
Use the ai-personal-stylist skill, but do not add anything to ~/.hermes/data/wardrobe.json. Treat this image only as inspiration/reference. Analyze the style aesthetic, clothing categories, color palette, mood, occasion fit, and how this look could be recreated in real life.
```

## 10. Does This Suit Me

```text
Use the ai-personal-stylist skill, but do not add anything to wardrobe. Treat this image as inspiration only. Based on my style profile, tell me whether this look suits me, what parts are most wearable, and what I should avoid copying literally.
```

## 11. Translate To Real Life

```text
Use the ai-personal-stylist skill, but do not add anything to wardrobe. Treat this image only as inspiration. Translate this look into a wearable real-life outfit with simpler alternatives and practical shopping suggestions.
```

## 12. Create 3 Inspired Variations

```text
Use the ai-personal-stylist skill, but do not add anything to wardrobe. Use this image only as a style reference. Create 3 inspired outfit directions: one casual, one polished, and one bold.
```

## 13. Color Palette Analysis

```text
Use the ai-personal-stylist skill, but do not add anything to wardrobe. Analyze the color palette in this image, explain why it works, and suggest 3 similar palettes I could use in real life.
```

## 14. Wardrobe Mode Opt-In

```text
Use the ai-personal-stylist skill. I am now sending a real clothing photo from my own wardrobe. Analyze it, and only after the analysis add it to ~/.hermes/data/wardrobe.json in the correct category.
```

## 15. Style Transition

```text
Use the ai-personal-stylist skill. Read my current wardrobe and build a 30-day transition plan toward [target style]. Split the advice into keep, restyle, and buy-later. Prioritize practical changes first.
```

## 16. Visual Transform (Reference Image -> New Outfit)

```text
Use the ai-personal-stylist skill. Take this reference image and transform it into a completely new outfit. Make it a smart-casual everyday look for Istanbul spring. Keep the color palette and mood, but change the silhouette and pieces entirely. Generate the transformed version as a new image.
```

## 17. Visual Transform - Office Version

```text
Use the ai-personal-stylist skill. Transform this reference look into a polished office-appropriate version. Keep the aesthetic energy but replace pieces with work-suitable alternatives. Generate the result as a new fashion image.
```

## 18. Visual Transform - Budget Version

```text
Use the ai-personal-stylist skill. Transform this reference look into a budget-friendly version under $150 total. Keep the vibe but use affordable, accessible alternatives. Generate the result as a new image.
```

## 19. Moodboard Analysis

```text
Use the ai-personal-stylist skill. I am sending multiple inspiration images. Analyze them together as a moodboard. Find the common aesthetic thread, shared palette, and recurring style elements. Then suggest 3 concrete outfit ideas that capture the unified mood.
```

## 20. Shopping Suggestions

```text
Use the ai-personal-stylist skill. Based on this outfit or look, find real products I can buy. Search for affordable alternatives and tell me where to find them with approximate prices.
```

## 21. Color Palette Check

```text
Use the ai-personal-stylist skill. Analyze the color palette in this image. Tell me the harmony type, undertones, and which seasonal color category it fits. Then suggest 3 alternative palettes I could use, and recommend specific clothing colors that pair well.
```

## 22. Try-On Visualization

```text
Use the ai-personal-stylist skill. I want to see how this outfit would look on me. Use my photo as reference and generate a visualization of me wearing the described outfit. Note that this is an approximation.
```

## 23. Quick Daily Outfit

```text
Use the ai-personal-stylist skill. Check today's weather for Istanbul. Based on my style profile, suggest one quick outfit for today. Keep it short and practical.
```
